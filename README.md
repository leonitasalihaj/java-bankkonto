# Java Bankkonto

Mein erstes Java-Projekt im Rahmen meines Computer-Science-Engineering-Studiums.

## Funktionen

- Bankkonto mit Inhaberin und Startguthaben erstellen
- Geld einzahlen
- Geld abheben
- Kontostand anzeigen
- Prüfen, ob genügend Guthaben vorhanden ist

## Verwendete Technologien

- Java
- IntelliJ IDEA
- Git
- GitHub

## Lernziele

- Klassen und Objekte
- Konstruktoren
- Methoden
- if-Abfragen
- Variablen und Datentypen
